Git Introduction

Distributed Version Control Systems

分散式版本控制系統 - Git, Mercurial, Bazaar or Darcs 每一個客戶端都擁有完整的先前版本資料 若伺服端資料丟失, 任一個客戶端, 都能回復先前版本的資料 各客戶在工作目錄進行修該時, 無需上網 客戶端的指令操作, 只需要近端資料庫, 因此速度極快 這裡所謂的 repository, 就是資料倉儲的意思 https://github.com/doggy8088/Learn-Git-in-30-days
